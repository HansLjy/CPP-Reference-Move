# C++ Reference Move

Rename a list of files in a C++ project and update the references automatically.

# Preliminary

1. Nodejs (version >= 23.6.1 preferred)

# Usage

1. Download and unzip / clone the repo.
2. In your terminal, type in
```shell
node DOWNLOAD_DIR/index.js SOURCE_DIR SOURCES DESTINATION --cmd-path=CMD_PATH --git
```
The parameters:
    a. `DOWNLOAD_DIR` is the download directory of the repo.
    b. `SOURCE_DIR` is the root of the source directory. The script will only search for the C++ files under this directory (recursively).
    c. `SOURCES` and `DESTINATION` follow the same rule as the `mv` command in linux-like systems. The paths can be either absolute or relative to where the command is executed. Wildcard allowed.
    d. `CMD_PATH` is the path to the `compile_commands.json` [generated by CMake](https://cmake.org/cmake/help/latest/variable/CMAKE_EXPORT_COMPILE_COMMANDS.html). If the option is not provided, the script will search for `compile_commands.json` under `build/`
    e. `--git` option is for working with `git` repositories. When enabled, `mv` command will be replaced by `git mv`. Make sure the command is executed inside the correct git repository.
